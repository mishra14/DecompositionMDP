static void mdp.util.PlanarSeparator.DFSDecomposition ( String  initialState,
MDP  mdp,
int  region 
)
static

This method creates DFS based decomposition of the given MDP.

This method creates a decomposition by using a count based heuristic. Starting from the supplied initial State Label, the method places each newly discovered State into the current Region.
When the State count inthe current region exceeds the total of Vertex Count / Region Count, then the Region count is incremented.
This process is continued till all the states are explored via DFS and placed in a region. The final decomposition is placed in the Layers.

static Map<String, LinkedHashSet<String> > mdp.util.PlanarSeparator.improveDecomposition ( MDP  mdp)
static

This method improves the basic DFS/BFS based decomposition of the given MDP.

This method improves the decomposition by using an heuristic that a State should belong to the Region To and From which it has the maximum number of Transition.
The method goes through each State in the nested Map that contains Key=State and Value=Map of {Key = Region and Value = Count of Transition to and From that Region to that State}
Then the resultant Map is checked to return the Dominant Region of every State and if the State does not belong to that Region then its Region is changed to that Region. Finally the resultant decomposition is returned to the calling method.
