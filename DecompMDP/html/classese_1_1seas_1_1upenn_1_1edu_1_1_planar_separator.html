static Map<String, LinkedHashSet<String> > ese.seas.upenn.edu.PlanarSeparator.improveDecomposition ( MDP  mdp)
static

This method improves the basic DFS/BFS based decomposition of the given MDP.

This method improves the decomposition by using an heuristic that a State should belong to the Region To and From which it has the maximum number of Transition.
The method goes through each State in the nested Map that contains Key=State and Value=Map of {Key = Region and Value = Count of Transition to and From that Region to that State}
Then the resultant Map is checked to return the Dominant Region of every State and if the State does not belong to that Region then its Region is changed to that Region. Finally the resultant decomposition is returned to the calling method.
