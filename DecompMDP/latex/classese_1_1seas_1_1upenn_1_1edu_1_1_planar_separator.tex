This method improves the basic D\+F\+S/\+B\+F\+S based decomposition of the given \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_m_d_p}{M\+D\+P}. 

This method improves the decomposition by using an heuristic that a \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_state}{State} should belong to the Region To and From which it has the maximum number of \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_transition}{Transition}.~\newline
 The method goes through each \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_state}{State} in the nested Map that contains Key=\hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_state}{State} and Value=Map of \{Key = Region and Value = Count of \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_transition}{Transition} to and From that Region to that \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_state}{State}\}~\newline
 Then the resultant Map is checked to return the Dominant Region of every \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_state}{State} and if the \hyperlink{classese_1_1seas_1_1upenn_1_1edu_1_1_state}{State} does not belong to that Region then its Region is changed to that Region. Finally the resultant decomposition is returned to the calling method.
